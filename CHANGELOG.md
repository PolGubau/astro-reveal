## [1.0.0] - 2026-03-03

### Added

- Initial release
- 9 animation directions (top, bottom, left, right, scale, diagonals)
- Speed presets (instant, fast, normal, slow)
- Easing presets (smooth, bounce, elastic, sharp, soft)
- Distance presets (small, medium, large)
- Stagger animations support
- `Reveal` component
- `RevealList` component
- Data attributes API
- IntersectionObserver-based triggering
- Astro View Transitions support
- `prefers-reduced-motion` support
- TypeScript types
- Comprehensive documentation
