# Changesets

Este directorio contiene los changesets para gestionar versiones y changelog automáticamente.

## Cómo usar

1. Cuando hagas cambios, crea un changeset:

   ```bash
   npm run changeset
   ```

2. Selecciona el tipo de cambio (major, minor, patch)

3. Describe el cambio

4. Commit el changeset junto con tus cambios

5. Al hacer merge a `main`, se creará automáticamente un PR con la nueva versión
